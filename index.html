Wrong way to open KUFI ERP

You opened index.html directly.
This app must be served from a web server (GitHub Pages, or the local Python server).
Opening directly as a file:// link blocks localStorage.
Your data is safe — it's stored at localhost:8765.

✅ Correct way to start
Step 1 — One-time setup (first time only)
Open Terminal and run:
chmod +x ~/Downloads/KUFI\ ERP/erp/START_SERVER.command
Step 2 — Every time you want to open ERP
Double-click START_SERVER.command in the erp folder
Or open Terminal, cd to the erp folder, run python3 server.py then go to http://localhost:8765
